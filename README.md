# Responsive Landscape Landing Page 

This project is part of a step-by-step YouTube tutorial in which we build a responsive landscape landing page from scratch using HTML CSS & JavaScript.

### 🎬 [Watch the Demo & Code](https://youtu.be/fCNVu3gELoo) 

![preview img](/preview.png) 

### 📌 Project Features
- Responsive Landscape Landing Page Using HTML CSS JavaScript | Mobile First
- Clear and semantic HTML structure.
- Use of CSS variables and modern JavaScript functions.
- Animated moving images.
- Developed first with the Mobile First methodology, then for desktop.
- Compatible with all mobile devices and with a beautiful and pleasant user interface.
